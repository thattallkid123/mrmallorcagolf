# Mr Mallorca Golf

Next.js 14 App Router site for mrmallorcagolf.com

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Stack
- Next.js 14 (App Router)
- Plain CSS (globals.css + CSS variables)
- Deployed on Vercel via GitHub

## Structure
- `src/app/` — pages (each folder = a route)
- `src/components/` — shared Nav, Footer
- `src/styles/globals.css` — all CSS variables and shared styles
- `public/` — static assets (images, favicon)

## Editing
Edit any page file and push to GitHub — Vercel auto-deploys.
