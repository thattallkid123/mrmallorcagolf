# Translation Notes

Working notes gathered during the i18n migration.

## Brand and tone

- English source copy is premium, calm, direct, and personal.
- German, Spanish, French, Dutch, and Swedish pages currently lean more formal than English.
- Chinese copy should stay concise and practical rather than overly literal.

## Golf vocabulary to keep consistent

- `green fee` is often left as-is instead of fully translated.
- `PGA Advanced Professional` should remain in English as a credential.
- `Trackman Master` should remain branded and untranslated.
- `handicap` is usually kept as the standard golf term.
- `tee time` may be translated or localized, but should be used consistently within each locale.
- `on-course coaching` needs one preferred phrase per language and should not drift page to page.

## Content rules to enforce later

- Non-English routes should not contain fallback English body copy unless explicitly intended.
- Metadata and CTA copy should use the same tone as body content for each locale.
- Numbers, currency, and separators should match locale expectations.
- Golf course names remain unchanged.

## Suggested future glossary areas

- package names
- CTA labels
- golf terminology
- travel and concierge language
- coaching and improvement language
